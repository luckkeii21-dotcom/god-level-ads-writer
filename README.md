# God-Level Ads Writer — Claude Code Skill

A professional skill for writing every type of ad: short-form UGC video scripts, founder-led ads, testimonial ads, static image ads, Mini VSL, full VSL, influencer scripts, and complete Meta/TikTok/YouTube copy — for any industry, any funnel stage.

## What it writes

- Short-form video scripts (30–60s) for Meta, TikTok, YouTube
- UGC, founder-led, testimonial, faceless/AI-generated ads
- Mini VSL (2–5 min) and full Video Sales Letters
- Static image ad copy and carousel ads
- Complete Meta ad copy (primary text, headline, description, CTA)

## Works for

Lead generation, high-ticket sales, DTC, ecommerce, service businesses, med spa, dental, coaching, dealerships, wellness, SaaS, AI products, and more.

## Installation

### Option 1 — Install from this repo (recommended)

In your Claude Code terminal:

```bash
claude skill install https://github.com/luckkeii21-dotcom/god-level-ads-writer-skill
```

### Option 2 — Manual install

1. Clone or download this repo
2. Copy the `god-level-ads-writer` folder into your project's `.claude/skills/` directory:

```
your-project/
└── .claude/
    └── skills/
        └── god-level-ads-writer/
            ├── SKILL.md
            └── references/
```

3. Restart Claude Code

## How to use

Once installed, just describe what you need — the skill auto-triggers on any ad writing request:

> *"Write me a UGC ad script for my med spa"*
> *"Give me a hook for my coaching offer"*
> *"Write a VSL for my online course"*
> *"Write a static ad for my dental clinic"*

Or call it directly:

```
/god-level-ads-writer
```
